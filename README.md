## :pencil: About
Sample code repository to learn about **[Vue.js](https://vuejs.org/)** with tutorials.  
This repo is based on `Laravel` with vue scaffolding and including vue, vuex, vue-router.  



## :pushpin: Environments
- Laravel `7.22.4`
- @vue/cli `4.4.6`
- vuex `3.5.1`
- vue-router `3.4.3`
